# Документация: приложение «Материалы» (materials)

Документ описывает модели и разделы админ-панели для страницы «Материалы» в проекте PM.Meetup.

---

## 1. Назначение приложения

Приложение **materials** отвечает за:

- хранение **категорий материалов** (кнопки-фильтры над карточками: «Курсы», «Хендбуки», «Фото с мероприятий» и т.д.);
- хранение самих **материалов** (карточки с обложкой, датой, местом проведения/форматом, длительностью, счётчиком просмотров);
- выдачу данных для отдельной страницы «Материалы» и API-фильтрации по категории/поисковой строке.

Все модели используют **TimeStampedModel** из приложения `core`.

---

## 2. Модели

### 2.1. MaterialCategory (категория материалов)

Категории соответствуют кнопкам над списком карточек на макете: «КУРСЫ», «ХЕНДБУКИ», «ФОТО С МЕРОПРИЯТИЙ», «ЗАПИСИ ВЫСТУПЛЕНИЙ», «ПРЕЗЕНТАЦИИ», «КНИГИ».

- **slug** — SlugField(50), уникальный. Машинное имя категории, используется в URL и фильтрации API.
- **title** — CharField(100). Название категории в админке и на сайте.
- **display_order** — PositiveIntegerField. Порядок отображения кнопок категорий.
- **is_active** — BooleanField. Показывать ли категорию на сайте.
- **created_at, updated_at** — DateTimeField. Авто-поля TimeStampedModel.


Рекомендуемые slug'и:

- `courses` — Курсы  
- `handbooks` — Хендбуки  
- `event_photos` — Фото с мероприятий  
- `talk_records` — Записи выступлений  
- `presentations` — Презентации  
- `books` — Книги

### 2.2. Material (материал)

Отдельная карточка на странице «Материалы».

- **label** — CharField(50), null, blank. Верхний лейбл на карточке: «КУРС», «ЗАПИСЬ», «ХЕНДБУК», «ФОТОГРАФИИ».
- **title** — CharField(300). Название материала (нижний заголовок на карточке).
- **category** — FK → MaterialCategory. Категория, к которой относится материал (фильтрация по кнопкам).
- **date** — DateField, null, blank. Дата мероприятия/материала (левая часть верхней строки).
- **place** — CharField(100), blank. Место или формат: «Минск», «Online» и т.п. (правая часть верхней строки).
- **duration_minutes** — PositiveIntegerField, null, blank. Число минут (жёлтая цифра на карточке).
- **description** — TextField, blank. Расширенное описание (для детальной страницы / всплывашки).
- **file_url** — URLField, blank. Ссылка на страницу/файл: запись, презентация, хендбук и т.д.
- **cover_image** — ImageField, blank, null. Обложка карточки (media/materials/).
- **view_count** — PositiveIntegerField, default=0. Счётчик просмотров материала.
- **created_at, updated_at** — DateTimeField. Авто-поля TimeStampedModel.


---

## 3. Админ-панель

- **Категории материалов** (`MaterialCategory`)  
  - список: название, slug, порядок, активность, дата создания;  
  - редактируемые в списке: порядок и флаг активности;  
  - slug автоматически подставляется из названия.
- **Материалы** (`Material`)  
  - список: название, лейбл, категория, дата, место/формат, длительность, просмотры, дата создания;  
  - фильтры: по категории и дате;  
  - поиск: по названию, лейблу и месту;  
  - обложка и ссылка (`cover_image`, `file_url`) задаются в форме редактирования.

## 4. Краткая сводка для заказчика


| Вопрос                                                            | Ответ                                                                                                          |
| ----------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| Где заводятся кнопки «Курсы / Хендбуки / Фото с мероприятий / …»? | В разделе «Категории материалов» (`MaterialCategory`), поля «Название» и «Порядок отображения».                |
| Где управлять карточками на странице «Материалы»?                 | В разделе «Материалы» (`Material`): указываются категория, дата, место/формат, длительность, обложка и ссылка. |
| Можно ли временно скрыть категорию?                               | Да, достаточно снять флаг «Активна» у нужной категории — кнопка исчезнет с сайта.                              |
| За что отвечает жёлтое число минут на карточке?                   | Это поле `duration_minutes`. Если его не заполнять, число не обязательно выводить на фронтенде.                |
