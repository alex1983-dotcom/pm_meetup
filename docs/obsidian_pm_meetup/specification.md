**Техническое задание на разработку админ-панели сайта PM.Meetup**

**Функциональные требования**

**Аутентификация и авторизация**

**Система входа**

1. Вход по email и паролю

2. Восстановление пароля через email

3. Автоматический выход при неактивности (настраиваемый таймаут)

4. Логирование всех попыток входа (опционально)

 **Роли и права доступа**

| Роль | Права доступа | Ограничения |
| :---- | :---- | :---- |
| Суперадминистратор | Полный доступ ко всем разделам | Нет |
| Администратор | Управление контентом, пользователями, событиями | Нет доступа к настройкам системы |
| Модератор | Модерация комментариев, заявок | Только чтение основных разделов |
| Организатор | Создание и редактирование своих событий | Нет доступа к чужим событиям |

**Управление событиями (митапами)**

**Создание события**

1. Название события (обязательное поле, до 100 символов)

2. Описание события (rich text editor с поддержкой markdown)

3. Дата и время начала 

4. Дата и время окончания (опционально)

5. Место проведения:

   1. Офлайн: адрес, город, помещение, карта (интеграция с картами)

   2. Онлайн: ссылка на трансляцию, платформа

   3. Гибридный формат

6. Тип события: митап, воркшоп, конференция, нетворкинг

7. Категория/тематика (множественный выбор из списка)

8. Обложка события (загрузка изображения, рекомендуемый размер 1200x630px)

9. Ограничение количества участников

10. Регистрация: открытая/требует подтверждения/закрытая

11. Теги для поиска

12. SEO-поля: meta title, meta description, URL slug

 **Управление расписанием события**

1. Добавление секций/сегментов программы

2. Время начала и окончания каждой секции

3. Название сегмента, описание

4. Привязка спикеров к сегменту

5. Место проведения сегмента (для многозальных событий)

6. Изменение порядка сегментов (drag-and-drop)

**Управление спикерами** 

1. Создание профиля спикера:

   1. ФИО

   2. Должность и компания

   3. Фотография

   4. Краткая биография

   5. Контакты (email, социальные сети)

   6. Темы выступлений

2. Привязка спикера к событию

3. База спикеров для повторного использования

 **Статусы события**

1. Черновик — событие создано, но не опубликовано

2. Опубликовано — доступно для регистрации

3. Регистрация закрыта — событие видно, но регистрация недоступна

4. Завершено — событие прошло

5. Отменено — событие отменено с указанием причины

**Действия над событиями**

1. Создать новое событие

2. Редактировать существующее

3. Дублировать событие (для серии митапов)

4. Изменить статус

5. Удалить событие (с подтверждением)

6. Просмотр на сайте (preview)

7. Экспорт списка участников (CSV, Excel)

 **Управление пользователями**

**Список пользователей**

1. Таблица с информацией:

   1. ID пользователя

   2. ФИО

   3. Email

   4. Телефон

   5. Дата регистрации

   6. Статус (активен/заблокирован)

   7. Роль

   8. Количество посещённых событий

2. Фильтрация: по роли, статусу, дате регистрации

3. Поиск: по имени, email, телефону

4. Сортировка по любому столбцу

5. Пагинация (50/100/200 записей на странице)

**Профиль пользователя**

1. Просмотр полной информации о пользователе

2. Редактирование данных

3. Смена пароля (для администратора)

4. Изменение роли

5. Блокировка/разблокировка учётной записи

6. История активности:

   1. Зарегистрированные события

   2. Посещённые события

   3. Оставленные комментарии

   4. История изменений профиля

 **Массовые действия**

1. Выбор нескольких пользователей (чекбоксы)

2. Экспорт выбранных (CSV, Excel)

3. Отправка email-рассылки выбранным

4. Массовое изменение статуса

5. Массовое удаление

**Управление регистрациями на события**

**Список регистраций**

1. Фильтр по событию

2. Информация о регистрации:

   1. ФИО участника

   2. Email, телефон

   3. Дата регистрации

   4. Статус: подтверждена/ожидает подтверждения/отменена

   5. Статус посещения: посетил/не явился/неизвестно

   6. Дополнительные поля (если были в форме регистрации)

3. Фильтрация по статусу

4. Поиск по имени/email

5. Экспорт списка участников

**Действия с регистрациями**

1. Подтверждение регистрации

2. Отклонение регистрации (с указанием причины)

3. Отправка письма участнику

4. Отметка о посещении события

5. Ручное добавление участника

6. Отмена регистрации

**Управление контентом сайта**

**Страницы**

1. Создание новых страниц (О нас, Контакты, Правила и т.д.)

2. Rich text editor с функциями:

   1. Форматирование текста

   2. Вставка изображений

   3. Вставка видео (embed)

   4. Таблицы

   5. Списки

   6. Ссылки

3. SEO-настройки для каждой страницы

4. Управление видимостью (опубликовано/черновик)

5. Настройка URL (slug)

**Блог/Новости**

1. Создание статей с возможностью:

   1. Заголовок и подзаголовок

   2. Контент (rich text)

   3. Обложка статьи

   4. Категория

   5. Теги

   6. Дата публикации (можно отложить)

   7. Автор статьи

2. Управление категориями блога

3. Управление тегами

**Медиабиблиотека**

1. Загрузка файлов: изображения, PDF, видео

2. Организация в папки

3. Поиск по названию файла

4. Фильтрация по типу файла

5. Просмотр свойств файла (размер, разрешение, дата загрузки)

6. Редактирование изображений: обрезка, изменение размера

7. Автоматическое создание превью

8. Массовая загрузка файлов

9. Удаление неиспользуемых файлов

**Партнёры и спонсоры**

1. Управление логотипами партнёров

2. Название компании, описание, ссылка на сайт

3. Уровень партнёрства (генеральный, золотой, серебряный и т.д.)

4. Привязка партнёра к конкретному событию или глобально

5. Порядок отображения на сайте

**Коммуникации**

**Email-рассылки**

1. Создание шаблонов писем:

   1. Подтверждение регистрации

   2. Напоминание о событии

   3. Благодарность за участие

   4. Новостная рассылка

2. Визуальный редактор писем

3. Тестовая отправка

4. Отправка по расписанию

5. Сегментация аудитории (по событиям, ролям, активности)

6. Статистика: открытия, клики, отписки

 **Уведомления (Push/In-app)**

1. Создание push-уведомлений

2. Планирование отправки

3. Целевая аудитория

4. Отслеживание доставки

 **Модерация комментариев**

1. Список всех комментариев на сайте

2. Статус: одобрен/на модерации/отклонён/спам

3. Модерация: одобрить, отклонить, пометить как спам

4. Редактирование комментария

5. Блокировка пользователя за нарушение правил

**Аналитика и отчёты**

**Дашборд (главная страница админ-панели)**

1. Виджеты с ключевыми метриками:

   1. Количество зарегистрированных пользователей (всего и за период)

   2. Количество предстоящих событий

   3. Количество регистраций на события

   4. Средняя посещаемость событий

   5. Новые регистрации за последние 7/30 дней

2. Графики:

   1. Динамика регистраций пользователей

   2. Посещаемость событий (по месяцам)

   3. Популярность категорий событий

3. Последние действия: новые регистрации, комментарии на модерации, предстоящие события

**Отчёты по событиям**

1. Общая статистика по событию:

   1. Количество регистраций

   2. Количество фактически посетивших

   3. Процент явки

   4. Демография участников (если собиралась)

2. Сравнение событий

3. Экспорт отчётов (PDF, Excel)

 **Отчёты по пользователям**

1. Активность пользователей

2. Самые активные участники

3. Вовлечённость (посещения, комментарии, обратная связь)

**Интеграция с внешней аналитикой**

1. Подключение Google Analytics

2. Подключение Яндекс.Метрика

3. Настройка целей и событий

**Настройки системы**

**Общие настройки**

1. Название сайта

2. Логотип и фавикон

3. Контактная информация (email, телефон, адрес)

4. Социальные сети (ссылки)

5. Часовой пояс

6. Язык интерфейса (русский/английский)

**Настройки событий**

1. Максимальное количество участников по умолчанию

2. Автоматическое закрытие регистрации (за X дней/часов до события)

3. Автоматическое изменение статуса события на "Завершено"

4. Шаблон email-уведомлений для событий

**Настройки регистрации пользователей**

1. Обязательные поля при регистрации

2. Подтверждение email (вкл/выкл)

3. Автоматическое одобрение регистраций

4. Политика конфиденциальности (текст или ссылка)

**Интеграции**

1. Интеграция с календарями (Google Calendar, iCal)

2. Интеграция с платёжными системами (если планируется платная регистрация)

3. Интеграция с сервисами рассылок (Mailchimp, SendGrid)

4. API для внешних приложений

5. Webhook для событий системы

**Безопасность**

1. Настройки паролей (минимальная длина, сложность)

2. Настройки сессий (время жизни)

3. Белый/чёрный список IP-адресов

4. Логирование действий администраторов

5. Резервное копирование (настройка расписания)

**Управление администраторами**

**Список администраторов**

1. Просмотр всех пользователей с правами администрирования

2. Фильтрация по роли

3. Информация: ФИО, email, роль, дата последнего входа

**Добавление/редактирование администратора**

1. ФИО

2. Email

3. Пароль (генерация или ручной ввод)

4. Назначение роли

5. Права доступа (детальная настройка, если нужна гибкость)

6. Активация 2FA

**Журнал действий (Audit Log)**

1. Логирование всех действий администраторов:

   1. Кто выполнил действие (пользователь, роль)

   2. Что было сделано (создание, редактирование, удаление)

   3. Какой объект был изменён (событие, пользователь, настройка)

   4. Дата и время действия

   5. IP-адрес

2. Фильтрация по дате, действию, администратору

3. Экспорт журнала
